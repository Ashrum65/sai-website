## 🎯 О сайте

Это репозиторий официального сайта проекта "Сай" (System of Adaptive Instruments).

**На сайте публикуется:**
- 📝 Статьи и заметки о философии проекта, ИИ, инженерии
- 👤 Информация об авторе и проекте
- 📚 Выдержки из технической документации (полная — в [SAI_project](https://github.com/ashrum65/SAI_project))
- 🔗 Ссылки на репозитории и ресурсы

**Не публикуется:**
- 🔒 Исходный код ядра системы
- 🔒 Конфиденциальные настройки и данные
- 🔒 Полная техническая документация (только в приватном репозитории)

### Почему так / Why this approach:
1. **Безопасность** / Security: Защита интеллектуальной собственности на ранней стадии
2. **Фокус** / Focus: Сайт — для широкой аудитории, репозиторий — для технической документации
3. **Гибкость** / Flexibility: Можно публиковать контент на сайте, не раскрывая всю систему

## Как обновляется / How it updates

При изменении документации в `SAI_project`:
1. Автор решает, какие части адаптировать для публичного доступа / The Author decides which parts to adapt for public access
2. Создается упрощенная версия для сайта / A simplified version is created for the website
3. Добавляется ссылка на полный репозиторий (для тех, у кого есть доступ) / A link to the full repository is added (for those with access)
